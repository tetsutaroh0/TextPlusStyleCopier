\# Text+ Style Copier for DaVinci Resolve（日本語版）



DaVinci Resolve用のPythonスクリプトです。

Media Pool / Power Bin で選択した Text+ クリップの\*\*スタイルを参照元として\*\*、

タイムライン上の複数の Text+ に一括適用できます。



クリップカラーで対象を絞り込むことで、効率的なテロップ・タイトル編集を実現します。



\---



\## ✨ 主な機能



\* 🎯 複数の Text+ にスタイルを一括適用

\* 🎨 クリップカラーによる対象フィルタリング

\* 🔁 ウィンドウ常駐（何度でも連続実行可能）

\* 🧠 前回設定（色・オプション）を記憶

\* 📝 テキスト本文は保持（スタイルのみコピー）

\* 🗑 参照用クリップは自動削除

\* 🎬 再生ヘッド位置を復元



\---



\## 🧰 動作環境



\* DaVinci Resolve（Studio版推奨）

\* Pythonスクリプトが有効になっている環境

\* Fusion UIManager が使用可能



> ⚠️ Resolve Free版ではUIが動作しない場合があります



\---



\## 🚀 インストール方法



1\. スクリプトを以下の名前で保存



```

TextPlusStyleCopier.py

```



2\. Resolveのスクリプトフォルダへ配置



\### Windows



```

C:\\\\ProgramData\\\\Blackmagic Design\\\\DaVinci Resolve\\\\Fusion\\\\Scripts\\\\Edit

```



\### Mac



```

/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility

```



3\. DaVinci Resolve を再起動



4\. 以下から実行



```

Workspace → Scripts → TextPlusStyleCopier_JP

```



\---



\## 🧑‍💻 使い方



\### ① スタイル参照元を選択



Media Pool または Power Bin で

\*\*Text+ クリップを1つ選択\*\*



👉 このクリップのスタイルがコピーされます



\---



\### ② 対象クリップに色を付ける



タイムライン上で、スタイルを適用したい Text+ に

\*\*同じクリップカラー（例: Orange）\*\* を設定



\---



\### ③ スクリプト実行



\* UIで対象カラーを選択

\* 必要に応じて：



&#x20; \* 「処理後もクリップカラーを残す」をON/OFF

\* 「Apply（実行）」をクリック



\---



\### ④ そのまま繰り返し可能



ウィンドウは閉じないため：



\* 参照元クリップを変更

\* 対象カラーを変更

\* すぐ再実行



が可能です



\---



\## 🧠 動作の仕組み



1\. Media PoolのText+をタイムライン末尾に一時追加

2\. FusionのTextPlus設定を取得

3\. 指定カラーのText+をすべて検索

4\. スタイルのみ適用（本文は維持）

5\. 一時クリップを削除

6\. 再生ヘッド位置を元に戻す



\---



\## ⚠️ 制限事項



\* Resolve APIの制限により

&#x20; 👉 タイムラインの「複数選択クリップ取得」は安定しない



→ そのため本スクリプトでは

\*\*クリップカラーで対象を指定する方式\*\*を採用しています



\---



\* 参照元クリップは一瞬タイムラインに追加されます

&#x20; （完全に非表示にはできません）



\---



\## 💡 使い方のコツ

&#x20; \* Orange → テロップ

&#x20; \* Blue → タイトル
